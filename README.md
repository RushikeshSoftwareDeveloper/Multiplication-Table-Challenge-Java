# Multiplication-Table-Challenge-Java
Console-based multiplication challenge game built in Java featuring random math questions, score tracking, and time-based gameplay.
# ⚡ Multiplication Table Challenge (Java)

## 📌 Project Description
A console-based math challenge game developed using Core Java.  
The system generates random multiplication questions, and the player must answer quickly to increase their score.  
A wrong answer results in game over.

## 🚀 Features
- Random multiplication questions
- Score tracking system
- Fast answer challenge
- Game over on wrong answer
- Interactive console gameplay

## 🛠 Concepts Used
✔ Random Number Generation  
✔ Loops (game loop)  
✔ Time-based challenge  
✔ Conditional Logic  
✔ Score System  

## 🎮 Game Rules
- The system generates a random multiplication question.
- Player must answer correctly to increase score.
- Wrong answer → Game Over.
- Game continues until a wrong answer is given.

## ▶ How to Run
1. Clone the repository
2. Open the project in Eclipse / IntelliJ
3. Run the main class
